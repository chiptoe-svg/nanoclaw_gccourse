A — Chat-first

One conversation owns the screen. Model picker, skill chips, cost annotations all live inside the chat surface. Persona is a slide-down drawer. Snapshots are a tray on the right. Closest to a real chat app. Best when iteration = "talk, tweak one thing, talk again."

B — Lab notebook

Two columns, always visible: configuration (model, skills, persona, session metrics) on the left; chat on the right. Knobs and chat coexist. Best when "what config did this come from?" needs to be answerable at every glance. Compare = open a second chat column.
