Agent Playground v3 — Brainstorm Mockups

Visual mockups produced during the brainstorming session on 2026-05-13. They informed the v3 design spec and the implementation plan. Listed in chronological order; later mockups generally override earlier ones. The canonical entries are the ones the spec reflects.

Mockups (chronological)

  1. 01-philosophy.htmlsuperseded
    Three competing surface philosophies (Chat-first / Lab notebook / Experiment dashboard). User rejected all three: "you're trying to do too much in one screen" — drove the move to a tab-per-concept structure.
  2. 02-waiting.htmlsuperseded
    Transition screen. "three tabs decided: Chat / Persona / Skills" (before Models was added as a fourth tab).
  3. 03-tab-structure.htmlsuperseded
    First full mockup of the 4-tab structure with draft banner. Established the overall shape.
  4. 04-three-panels.htmlsuperseded
    First three-panel layout (Library / Preview / Active) on Persona and Skills tabs, plus refined Chat with provider+model in toolbar, and Models tab with local-model cards.
  5. 05-corrected-layers.htmlcanonical
    Persona right-panel sub-tabs corrected to the actual file structure (My persona / Group base / Container base / Global, dimmed when absent) — backed by the architectural decision to add a host-side getEffectivePersonaLayers helper. Also: Models tab cards with parameter count, modalities, and instructor notes.
  6. 06-provider-warning.htmlcanonical
    Chat tab — provider switch confirmation modal ("⚠ Switch provider? — this will reset chat") and model switch inline pill ("↻ next message"). Captures the asymmetry between provider switch (kills container, resets chat) and model switch (env var only, chat continues).
  7. 07-persona-cleanup.htmlsuperseded
    Middle panel cleaned up — no magic Copy/Replace buttons. Standard select-and-copy with native browser behavior. (My-library inline × delete affordance was added and then rejected in 08.)
  8. 08-no-library-delete.htmlsuperseded
    Inline × delete affordance removed from My library entries — list-only listing across all three tiers. (Deletion handled separately, out of scope for v3.)
  9. 09-persona-final.htmlsuperseded
    Library width narrowed to 150px; middle panel re-labeled "📚 From library: …"; bottom-right "Test in Chat" button removed (chat always talks to live draft, so testing = clicking the Chat tab). "Where everything happens" recap table at the bottom.
  10. 10-correct-data.htmlsuperseded
    Demo data corrected to reflect the actual install — Felix is the current agent (right panel), concise_writer is what the user clicked in the library (middle panel). Added prefers provider dropdown alongside model.
  11. 11-banner-buttons.htmlcanonical
    Final Persona-tab layout. Draft-banner buttons get distinct color treatment (red Discard / green Save to my library / blue Apply to my agent). Right-panel footer reminder text removed. This is the layout the spec describes.
  12. 12-closing.htmlsuperseded
    Brainstorm-closing waiting screen.
  13. 13-plan-done.htmlsuperseded
    Plan-written closing screen.
